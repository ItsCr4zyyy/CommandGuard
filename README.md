# CommandGuard

This plugin hides unauthorized commands from the player's tab-completion (when typing `/`). Players will only see and be able to autocomplete the commands you explicitly allow in the configuration.

## ✅ Features
* Perfectly clean tab-completion (no `/bukkit:?`, `/pl`, or admin commands).
* Smart tab-completion for `/msg` (only suggests online players).
* No server restart required when changing settings.

## 🛠 Requirements

- **Minecraft 1.21.11 and above**
- **Java 21**

## 💬 Commands & Permissions
* `/cgreload` - Reloads the `config.yml` file. (Permission: `commandguard.reload`)
* `commandguard.bypass` - Players with this permission ignore the filter and can see all commands on the server.

## 📥 Installation

1. Download the plugin `.jar` file
2. Place the plugin into your server's `/plugins` folder
3. Start the server and configure `config.yml`
